
# Adventure Works – Sales Analysis Dashboard (Power BI)

## Project Overview
This project is an **interactive Sales Analysis Dashboard** built using the **Adventure Works dataset** in **Power BI**.
The dashboard provides a comprehensive view of sales performance across **products, customers, categories, regions, and time periods**, helping stakeholders make data-driven decisions.

---

## Tools & Technologies
- Power BI Desktop
- DAX
- Power Query (ETL)
- Adventure Works Sample Dataset

---

## Business Objectives
- Analyze overall sales and revenue performance
- Identify top-performing products and customers
- Track monthly and yearly sales trends
- Compare category and sub-category performance
- Understand customer demographics and buying behavior
- Monitor returns and profitability
